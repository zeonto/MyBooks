remove 删除数据

语法格式:

db.collection.remove(
   <query>,
   {
     justOne: <boolean>,
     writeConcern: <document>
   }
)

deleteOne、deleteMany 删除数据

这 2 个语法是在 3.2 版本开始提供的，对应 remove 语法是否使用 justOne 参数

语法格式:

db.collection.deleteOne(
   <query>
)
db.collection.deleteMany(
   <query>
)
