2.1 命令连接

命令连接

远程连接

格式：mongo ip或DNS:端口号/数据库名 -u user -p password

命令：bin/mongo 192.168.9.209:27017

如果没有密码就不需要指定用户名和密码，另外如果在IP:端口后不指定库的话，默认连接test库。
